Game Flappy Bird

Game Flappy Bird adalah sebuah game 2D arcade platformer kasual yang mengadaptasi mekanik klasik Flappy Bird, dikembangkan menggunakan Godot Engine dengan visual pixel art yang bersih dan familier. Pemain mengontrol seekor burung merah kecil yang harus terus mengepakkan sayapnya untuk menjaga ketinggian dan menavigasi celah sempit di antara pipa-pippa hijau yang muncul tanpa henti. Mengandalkan kontrol satu ketukan (one-tap control) yang sederhana namun menuntut konsentrasi tinggi, game ini menguji ketangkasan dan timing presisi pemain demi melewati setiap rintangan, menghindari benturan dengan pipa atau tanah, dan mencetak skor tertinggi yang ditampilkan secara real-time di layar.

Aplikasi Kasir Digital dibangun dalam satu file HTML menggunakan JavaScript vanilla tanpa framework atau library tambahan. Semua data mulai dari produk, transaksi, hutang, pengeluaran, hingga pemasukan manual disimpan secara lokal di browser menggunakan localStorage, sehingga aplikasi tetap berfungsi tanpa koneksi internet maupun server backend. Tampilan antarmuka dirender secara dinamis dengan cara meng-inject string HTML langsung ke DOM, dan seluruh interaksi pengguna ditangani melalui satu event listener terpusat menggunakan teknik event delegation. Pendekatan ini membuat aplikasi tetap ringan, mudah didistribusikan, dan dapat langsung dijalankan di browser mana pun tanpa proses instalasi.
